# DataCafé Analytics – Pipeline de Dados de Vendas

## 📌 Objetivo
Desenvolver um pipeline de dados completo simulando um ambiente de engenharia de dados, incluindo ETL, modelagem relacional e visualização.

## 🛠 Tecnologias utilizadas
- SQL (SQLite)
- Python (Pandas - Google Colab)
- Looker Studio (Dashboard)

## 🔄 Etapas do Projeto

### 1. Modelagem de Dados
Estrutura baseada nas 3 formas normais (1FN, 2FN e 3FN), com separação em:
- Clientes
- Produtos
- Vendas

### 2. ETL (Extração, Transformação e Carga)
- Simulação de dados brutos com inconsistências
- Tratamento de dados (tipos, valores inválidos)
- Carga controlada com JOIN garantindo integridade

### 3. Análise de Dados (SQL)
- Faturamento total
- Produto mais vendido
- Cliente com maior volume de compras
- Vendas por categoria e data

### 4. Visualização de Dados
Dashboard criado no Looker Studio contendo:
- Faturamento por produto
- Vendas por categoria
- Evolução de vendas por data

## 📊 Resultados

- Faturamento total: 137
- Produto mais vendido: Cappuccino
- Cliente destaque: Ana Costa

## 🚀 Diferenciais do Projeto
- Pipeline completo de dados (ETL)
- Aplicação de normalização (3FN)
- Integração entre SQL e Python
- Visualização de dados profissional
